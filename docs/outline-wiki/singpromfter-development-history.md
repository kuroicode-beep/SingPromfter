# SingPromfter 개발 히스토리

작성일: 2026-07-04  
기준 버전: `1.0.1+1`

## 01. 버전명 결정

전역 업데이트 규칙에 따라 이번 문서/위키 업데이트와 재빌드는 패치 업데이트로 분류한다.

- 이전 정식 릴리스: `v1.0.0`
- 이번 버전명: `v1.0.1`
- Flutter 버전 표기: `1.0.1+1`
- 사유: 기능 추가가 아닌 프로젝트 위키 갱신, 릴리스 문서 정합성 보정, 재빌드 검증

## 03. 현재 릴리스 상태

- 앱 버전: `1.0.1+1`
- 버전명: `v1.0.1`
- GitHub 기본 브랜치: `master`
- Windows 빌드 명령: `flutter build windows`
- 배포 폴더 기준: `dist/SingPromfter-v1.0.0/`

## 04. 검증 기준

정식 릴리스와 문서 패치 후 다음 명령을 기준으로 검증한다.

```powershell
flutter analyze
flutter test
flutter build windows
```

## 05. 남은 운영 메모

- `audioplayers` 6.x, `file_picker` 11.x 메이저 업데이트는 별도 브랜치에서 진행한다.
- 실제 mp3 재생, 파일 선택 다이얼로그, 백업/복원, 삭제 실행 취소는 수동 회귀 테스트가 필요하다.
- Outline 문서 원본은 `docs/outline-wiki/`에 유지하며, Outline 문서가 손상되면 이 원본으로 재게시한다.
