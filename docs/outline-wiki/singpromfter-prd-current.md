# SingPromfter PRD

작성일: 2026-06-28  
기준 버전: `1.0.0+1`  
대상: 저시력 사용자, 무대/연습 환경에서 가사를 크게 보며 반주를 재생해야 하는 사용자

## 01. 제품 개요

SingPromfter는 가사와 반주를 한 곡 단위로 관리하고, 저시력 사용자에게 충분히 큰 글자와 높은 대비의 프롬프터 화면을 제공하는 로컬 우선 노래 프롬프터 앱이다. Flutter 기반으로 작성되어 Windows, Android, iOS, macOS, Linux, Web 구조를 갖지만, 현재 핵심 사용 시나리오는 로컬 파일 접근이 가능한 네이티브/데스크톱 앱이다.

## 02. 사용자

- 저시력으로 일반 노래방/가사 화면을 보기 어려운 사용자
- 무대 또는 원거리 환경에서 가사를 크게 확인해야 하는 사용자
- txt 가사와 mp3 반주를 직접 준비해 사용하는 개인 사용자
- 여러 곡을 순서대로 예약하고 연습 또는 공연 흐름을 관리해야 하는 사용자

## 03. 목표

- 가사를 검정 배경과 흰 글씨의 고대비 화면에 크게 표시한다.
- 곡별로 가사 1개와 반주 0~3개를 연결해 관리한다.
- 반주 재생, 정지, 처음부터 재생, 위치 이동, 볼륨 조절을 제공한다.
- 예약 큐를 통해 다음 곡을 순차 재생할 수 있게 한다.
- 글자 크기, 줄 간격, 스크롤 속도, 글꼴, 굵게 옵션을 저장하고 재실행 시 복원한다.
- 전체화면 프롬프터에서 가사를 무대용으로 표시한다.

## 04. 주요 기능 범위

- 곡 등록: txt 가사 파일 필수, mp3 반주 0~3개 선택
- 곡 수정: 제목, 가사, 슬롯별 반주 교체
- 곡 삭제: 곡 메타데이터와 로컬 가사/반주 파일 삭제
- 곡 선택: 마지막 선택 곡과 곡별 마지막 반주 슬롯 복원
- 반주 재생: `audioplayers` 기반 재생/일시정지/정지/처음부터/시크/볼륨
- 예약 큐: 예약 추가, 개별 삭제, 전체 비우기, 곡 종료 후 다음 예약곡 자동 재생
- 프롬프터: 전체화면, 자동 스크롤, 수동 위/아래 이동, 컨트롤 표시/숨김
- 접근성: 표준/저시력/원거리 프리셋, 글자 크기, 줄 간격, 속도, 글꼴, 굵게
- 로컬 저장: SharedPreferences와 앱 문서 디렉터리 파일 저장

## 05. 비범위

- 사용자 계정, 로그인, 클라우드 동기화
- 온라인 백업 또는 원격 저장소 연동
- 반주와 가사의 타임코드 동기화
- 가사 직접 편집 전용 에디터
- 예약 큐 드래그 재정렬
- TTS, AI 가사 생성, 블루투스 리모컨
- DRM 음원 또는 스트리밍 음원 연동

## 06. 성공 기준

- `flutter analyze`에서 이슈가 없어야 한다.
- `flutter test`가 통과해야 한다.
- Windows 네이티브 빌드가 성공하고 `singpromfter_app.exe`가 생성되어야 한다.
- 곡 등록 후 앱 재실행 시 곡 목록, 설정, 큐, 마지막 선택 곡이 복원되어야 한다.
- 저시력 프리셋과 원거리 프리셋 적용 시 가독성이 즉시 바뀌어야 한다.
- 반주 파일이 없거나 손상된 경우 사용자에게 명확한 안내를 제공해야 한다.

## 07. 리스크

- Web 플랫폼은 로컬 반주 파일 복사와 재생 준비에 제한이 있어 네이티브 앱과 동등하지 않다.
- 파일명 기반 저장 구조는 제목 변경, 동일 제목, 운영체제별 파일명 제한에 민감하다.
- mp3 외 오디오 파일 확장자/코덱 호환성은 플랫폼별 `audioplayers` 지원 범위에 영향을 받는다.
- 현재 테스트는 placeholder 수준이라 저장소, 파일 복사, 큐, 재생 상태에 대한 자동화 검증이 부족하다.
- README 기능 버전 `v0.3.1`과 `pubspec.yaml` 앱 버전 `1.0.0+1`의 버전 체계가 다르다.
