# SingPromfter 기능상세리스트

**문서 버전**: 1.0  
**작성일**: 2026-03-14  
**앱 버전**: 1.0.1+1

---

## 1. 개요

SingPromfter는 저시력 친화형 가사 프롬프터 애플리케이션입니다.  
가사(txt)와 반주(mp3) 파일을 등록·관리하고, 재생과 동기화된 자동 스크롤로 가사를 표시합니다.

---

## 2. 기능 분류 체계

| 대분류 | 중분류 | 설명 |
|--------|--------|------|
| 곡 관리 | 곡 등록 | 가사·반주 파일 추가 |
| 곡 관리 | 곡 수정 | 제목·가사·반주 변경 |
| 곡 관리 | 곡 삭제 | 곡 및 관련 파일 삭제 |
| 재생 | 반주 재생 | MR 1~3 슬롯 선택 및 재생 |
| 재생 | 예약 큐 | 곡 예약 및 순차 재생 |
| 표시 | 가사 표시 | 메인 패널 가사 뷰 |
| 표시 | 전체화면 프롬프터 | 풀스크린 가사 모드 |
| 접근성 | 시각 조정 | 글자 크기·줄간격·폰트·굵게 |
| 접근성 | 프리셋 | 표준/저시력/원거리 |
| 데이터 | 저장·복원 | 곡·설정·큐·마지막 선택 곡 |
| 데이터 | 마이그레이션 | 레거시 경로 호환 |

## 3. 기능 상세

### 3.1 곡 관리

#### 3.1.1 곡 등록

| 항목 | 내용 |
|------|------|
| **기능 ID** | F-SONG-001 |
| **설명** | 가사(txt)와 반주(mp3) 파일을 선택해 새 곡을 등록합니다. |
| **입력** | • 가사: txt 파일 1개 (필수)<br>• 반주: mp3 파일 0~3개 (선택, 슬롯 1~3) |
| **처리** | • 곡 제목: 파일명에서 .txt 제거 또는 사용자 입력<br>• 가사: UTF-8/CP949/latin1 디코딩 시도<br>• 반주: 앱 데이터 폴더로 복사, `{제목}_mr{n}.mp3` 형식<br>• 파일명: `<>:"/\|?*` 등 특수문자 제거 |
| **제약** | • 동일 제목 곡 중복 불가<br>• 웹: 반주 파일 복사 미지원(가사만 등록) |
| **출력** | 곡 목록에 추가, 해당 곡 자동 선택 |

#### 3.1.2 곡 수정

| 항목 | 내용 |
|------|------|
| **기능 ID** | F-SONG-002 |
| **설명** | 기존 곡의 제목·가사·반주를 수정합니다. |
| **입력** | • 제목: 텍스트<br>• 가사: 새 txt 파일 선택 또는 기존 유지<br>• 반주: 슬롯별 새 mp3 선택 또는 기존 유지 |
| **처리** | • 제목 변경 시 가사·반주 파일명 재생성<br>• 기존 반주 파일명이 새 제목과 맞으면 유지, 아니면 복사 후 기존 삭제 |
| **제약** | • 동일 제목(다른 곡) 중복 불가 |
| **출력** | 곡 정보 갱신, 선택 중이면 반주 재로드 |

#### 3.1.3 곡 삭제

| 항목 | 내용 |
|------|------|
| **기능 ID** | F-SONG-003 |
| **설명** | 곡과 관련 가사·반주 파일을 삭제합니다. |
| **입력** | 삭제 확인 다이얼로그 |
| **처리** | • 가사 파일 삭제 (현재 경로 및 레거시 경로)<br>• 반주 파일 삭제 (현재·레거시 mr 폴더)<br>• 예약 큐에서 해당 곡 제거 |
| **출력** | 목록에서 제거, 선택 중이면 다음 곡 선택 또는 선택 해제 |
