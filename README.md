# SingPrompt

노래 가사를 **초대형 글씨**로 표시하고 **MR(반주)**과 함께 재생할 수 있는 **저시력 친화 가사 프롬프터** 웹 앱입니다.

## 한 줄 정의

> SingPrompt는 노래 가사를 초대형 글씨로 표시하고 MR과 함께 재생할 수 있는 저시력 친화 가사 프롬프터 시스템이다.

## 기능

- **가사 업로드**: txt 파일 업로드
- **MR 업로드**: mp3 파일 업로드
- **가사 표시**: 전체화면 프롬프터, 검은 배경 + 흰 글씨
- **수동 스크롤**: 프롬프터 화면에서 위/아래 스크롤
- **MR 재생**: 재생 / 일시정지 / 정지 / 볼륨 조절
- **노래 목록**: 좌측 패널에서 곡 선택, 재생, 삭제
- **접근성**: 글자 크기·줄 간격 조절, 노란 강조 버튼

## 사용 방법

1. **실행**: `index.html`을 브라우저에서 열기 (로컬 서버 권장)
2. **가사 추가**: "가사(txt) + MR(mp3) 추가"로 txt 업로드 → 제목 입력
3. **MR만 추가**: 곡 선택 후 "MR(mp3) 추가"로 mp3 업로드
4. **재생**: 목록에서 곡 클릭 → 하단 재생 버튼으로 MR 재생
5. **프롬프터**: "전체화면 프롬프터" 클릭 → 검은 화면에 큰 글씨로 가사 표시, ▲▼로 스크롤
